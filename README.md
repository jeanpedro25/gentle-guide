# PROFETA BET - Previsões Inteligentes com IA

Sistema premium de previsão esportiva utilizando modelos matemáticos de Poisson, Kelly Criterion e Inteligência Artificial para análise de valor esperado (EV).

## 🚀 Tecnologias
- **Frontend**: React + Vite + TypeScript
- **Estilização**: Tailwind CSS + shadcn/ui
- **Backend**: Supabase (Auth, Database, Edge Functions)
- **Análise**: Google Gemini 3 Flash / 1.5 Flash
- **Dados**: iSports API Proxy

## 🛠️ Como rodar localmente

1. **Clonar o repositório**:
   ```sh
   git clone https://github.com/jeanpedro25/gentle-guide
   cd gentle-guide
   ```

2. **Instalar dependências**:
   ```sh
   npm install
   ```

3. **Configurar variáveis de ambiente**:
   Crie um arquivo `.env` com suas chaves do Supabase.

4. **Iniciar servidor de desenvolvimento**:
   ```sh
   npm run dev
   ```

## 📈 Funcionalidades
- **Análise Profunda**: Veredito de IA com confiança e justificativa tática.
- **Jogue Agora**: Ranking Top 10 de oportunidades nas próximas 4 horas.
- **Gestão de Banca**: Calculadora Kelly Criterion integrada.
- **Histórico Central**: Acompanhamento de Greens e Reds com resolução automática.
